To Do's Web App

This is simple web app made with HTML5, CSS and Vanilla Javascript, this app allows you to add, edit, delete and mark tasks as completed.
This project was created as practice to help me learn the Javascript fundamentals and improve my web development skills and logic.


FEATURES:

-Add new tasks using the input field or the enter key.
-Mark tasks as completed (with visual feedback).
-Edit existing tasks by clicking on their text.
-Delete Individual tasks using their separe button or delete the whole task list.
-Tasks are automatically saved in Local Storage so they remain even the page is reloaded.

TECHNOLOGIES USED:
-HTML5 - For Page structure.
-CSS3 - For Styling, design, and responsive layout.
-JAVASCIPT (ES6+) - For Logic, interactivity and local storage.
